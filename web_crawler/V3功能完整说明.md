## 一、核心功能

### 1.1 多格式文档爬取支持

不再仅限于PDF！现在支持**25种文档格式**：

#### 办公文档
| 类型 | 扩展名 | 说明 |
|------|--------|------|
| PDF | .pdf | Adobe PDF文档 |
| Word | .doc, .docx, .dot, .dotx, .rtf | Word文档及模板 |
| Excel | .xls, .xlsx, .xlsm, .xlsb, .csv | 电子表格 |
| PowerPoint | .ppt, .pptx, .pps, .ppsx | 演示文稿 |
| WPS | .wps, .et, .dps | WPS办公文档 |

#### 压缩包
| 类型 | 扩展名 | 说明 |
|------|--------|------|
| ZIP | .zip | ZIP压缩包 |
| RAR | .rar | RAR压缩包 |
| 7Z | .7z | 7-Zip压缩包 |
| TAR | .tar, .gz, .bz2 | TAR压缩包 |

#### 其他
| 类型 | 扩展名 | 说明 |
|------|--------|------|
| 文本 | .txt | 纯文本文档 |

#### 智能分类
下载的文档会自动按类型分类保存：
```
downloads/documents/
├── PDF/               # PDF文档
├── Word/              # Word文档
├── Excel/             # Excel表格
├── PowerPoint/        # PPT演示
├── Archive/           # 压缩包
├── Text/              # 文本文件
└── Other/             # 其他类型
```

#### 智能识别
- **格式识别:** 根据文件扩展名自动识别类型
- **比赛相关判断:** 根据文件名和链接文本判断是否比赛相关
- **智能命名:** 自动清理文件名，保留中文和关键信息

### 1.2 自动删除无效网址

#### 工作原理

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  URL访问                                            │
│     ↓                                               │
│  成功? ──→ 是 ──→ 清除失败记录 ──→ 继续            │
│     ↓                                               │
│    否                                               │
│     ↓                                               │
│  失败次数+1 ──→ 保存到记录文件                     │
│     ↓                                               │
│  检查自动删除设置                                   │
│     ↓                                               │
│  禁用? ──→ 是 ──→ 仅记录，不删除                  │
│     ↓                                               │
│    否                                               │
│     ↓                                               │
│  失败次数 >= 3?                                     │
│     ↓                                               │
│  是 ──→ 从urls.csv删除 ──→ 清理记录              │
│     ↓                                               │
│    否 ──→ 保留，下次继续尝试                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### 特点
- **智能记录:** 每个网址的失败次数单独记录
- **持久化存储:** 记录保存在 `urls_fail_records.json`
- **自动清理:** 成功访问后自动清除失败记录
- **可配置:** 支持调整重试次数或禁用自动删除

#### 使用场景
- **首次运行:** 自动清理404/403/无法访问的网址
- **定期维护:** 保持网址列表清洁
- **网络波动:** 临时失败不会立即删除

## 二、使用方式

### 2.1 GUI模式（推荐）

```bash
python app_gui.py
```

操作步骤：
1. 点击"设置"标签页
2. （可选）勾选"使用代理IP"并配置
3. 选择爬取模式
4. 点击"开始爬取"
5. 查看实时进度和结果

#### 模式选择
| 模式 | 说明 |
|------|------|
| 全部(文档+报告) | 下载所有文档 + 生成HTML/Word报告 |
| 仅下载文档 | 只下载比赛相关文档 |
| 仅生成报告 | 只生成HTML/Word报告 |

### 2.2 命令行模式

#### 基础使用
```bash
# 使用默认设置
python enhanced_crawler_v3.py --urls urls.csv

# 指定输出目录
python enhanced_crawler_v3.py --urls urls.csv --output ./my_output

# 设置超时时间
python enhanced_crawler_v3.py --urls urls.csv --timeout 60
```

#### 模式选择
```bash
# 仅下载文档
python enhanced_crawler_v3.py --urls urls.csv --mode doc

# 仅生成报告
python enhanced_crawler_v3.py --urls urls.csv --mode html

# 全部（默认）
python enhanced_crawler_v3.py --urls urls.csv --mode all
```

#### 代理配置
```bash
# 使用默认代理文件
python enhanced_crawler_v3.py --urls urls.csv --proxy

# 指定代理文件
python enhanced_crawler_v3.py --urls urls.csv --proxy --proxy-file my_proxies.json
```

#### 自动删除配置
```bash
# 设置5次失败后删除（默认3次）
python enhanced_crawler_v3.py --urls urls.csv --max-retries 5

# 完全禁用自动删除
python enhanced_crawler_v3.py --urls urls.csv --no-auto-remove
```

#### 完整示例
```bash
# 使用代理，5次失败后删除，仅下载文档
python enhanced_crawler_v3.py \
  --urls urls.csv \
  --mode doc \
  --proxy \
  --proxy-file proxies.json \
  --max-retries 5 \
  --output ./output
```

## 七、故障排除

### 问题1: 某些文档格式无法识别
**解决:** 检查文件扩展名是否在支持列表中（25种格式）

### 问题2: 有效网址被误删
**解决:** 
1. 恢复urls.csv备份
2. 使用 `--no-auto-remove` 禁用自动删除
3. 使用 `--max-retries 5` 增加重试次数

### 问题3: 代理不生效
**解决:**
1. 检查代理格式是否正确
2. 使用GUI界面的"测试代理"功能
3. 查看日志中的代理连接信息

### 问题4: 文档下载失败
**解决:**
1. 检查网络连接
2. 查看日志中的错误信息
3. 增加超时时间 `--timeout 60`
