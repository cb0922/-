## 🚀 快速开始 (3步)

### 第1步：安装 Python
- **要求**：Python 3.8 或更高版本
- **下载**：https://www.python.org/downloads/
- **⚠️ 重要**：安装时勾选 **"Add Python to PATH"**

### 第2步：一键安装依赖
- 双击运行 `一键安装.bat`
- 等待安装完成（约 5-10 分钟，取决于网速）
- 安装成功后会自动启动程序

### 第3步：开始使用
- 程序启动后，点击 **"开始爬取"** 按钮
- 等待爬取完成
- 查看生成的 Word 报告和下载的文档

## 📋 详细安装说明

### 方法一：自动安装 (推荐)

1. **确保 Python 已安装**
   ```cmd
   python --version
   ```
   应显示 Python 3.8.x 或更高版本

2. **双击运行安装脚本**
   ```
   一键安装.bat
   ```

3. **等待安装完成**
   - 脚本会自动检测 Python 环境
   - 自动安装所有依赖包
   - 使用国内镜像加速下载
   - 安装完成后自动启动程序

### 方法二：手动安装 (如果自动安装失败)

1. **打开命令提示符 (CMD)**
   - 按 `Win + R`，输入 `cmd`，回车

2. **进入程序目录**
   ```cmd
   cd /d C:\Users\你的用户名\Downloads\web_crawler
   ```

3. **安装依赖 (使用清华镜像)**
   ```cmd
   pip install -r requirements_gui.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

4. **启动程序**
   ```cmd
   python 启动器.py
   ```

### 方法三：离线安装 (无网络环境)

**在有网络的电脑上：**

1. **下载依赖包**
   ```cmd
   pip download -r requirements_gui.txt -d ./packages
   ```

2. **复制到目标电脑**
   - 将整个程序文件夹复制到目标电脑

3. **离线安装**
   ```cmd
   pip install --no-index --find-links=./packages -r requirements_gui.txt
   ```

## 🔧 常见问题解决

### Q1: 提示 "Python 未安装" 或 "python 不是内部命令"

**原因**：Python 未安装或未添加到 PATH

**解决**：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8+ 安装包
3. **⚠️ 关键**：安装时勾选 **"Add Python to PATH"**
   - 在安装界面底部找到 "Add Python to environment variables"
   - 勾选后点击 Install Now

### Q2: 安装过程中提示权限不足

**解决**：
- 右键点击 `一键安装.bat` → **"以管理员身份运行"**

### Q3: 下载速度慢/超时

**原因**：默认使用国外 PyPI 源

**解决** (已内置在脚本中)：
- 脚本已配置清华镜像源
- 如仍慢，可尝试其他镜像：
  - 阿里云：`https://mirrors.aliyun.com/pypi/simple`
  - 腾讯云：`https://mirrors.cloud.tencent.com/pypi/simple`

### Q4: 提示 "No module named 'PyQt6'"

**解决**：
```cmd
pip install PyQt6 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q5: 程序闪退/打不开

**排查步骤**：
1. **检查 Python 版本**
   ```cmd
   python --version
   ```

2. **检查依赖是否安装完整**
   ```cmd
   cd web_crawler
   python -c "import PyQt6; import aiohttp; import pandas; print('OK')"
   ```
   如有报错，重新运行安装脚本

3. **查看错误日志**
   - 打开命令提示符
   - 手动运行：`python 启动器.py`
   - 查看具体错误信息

### Q6: 安装后再次使用时如何启动？

**方法**：双击 `启动程序.bat`

或者：
```cmd
cd web_crawler
python 启动器.py
```

## 📖 使用说明

### 首次使用

1. **准备网址列表**
   - 编辑 `urls.csv` 文件
   - 格式：`网址,名称,分类`
   - 示例：
     ```
     url,name,category
     https://www.moe.gov.cn/,教育部,教育公告
     https://jyt.shanxi.gov.cn/,山西省教育厅,地方教育
     ```

2. **启动程序**
   - 双击 `启动程序.bat`

3. **开始爬取**
   - 在控制台标签页点击 **"开始爬取"**
   - 等待爬取完成（进度条显示实时进度）

4. **查看结果**
   - **Word报告**：点击"查看最新Word文档"
   - **下载的文档**：点击"打开文档文件夹"
   - **HTML报告**：点击"查看最新HTML报告"

### 日常使用

- **添加新网址**：在控制台输入网址，点击"添加"
- **导入全国网址库**：点击"导入全国网址库"按钮
- **定时爬取**：在"定时任务"标签页设置自动爬取
