========================================
    核心程序文件
========================================

启动器.py                    - 智能启动脚本
app_gui.py                   - GUI主程序
enhanced_crawler_v3.py       - 命令行版爬虫核心
word_generator.py            - Word报告生成器
report_generator_v3.py       - HTML报告生成器
core/document_handler.py     - 文档下载处理器
core/anti_detection.py       - 反检测模块
core/fetcher.py              - 网络请求模块
core/parser.py               - 内容解析模块
core/url_manager.py          - 网址管理模块
core/proxy_manager.py        - 代理管理模块
config/settings.py           - 配置文件
storage/database.py          - 数据存储模块

========================================
    依赖配置
========================================

requirements.txt             - 爬虫核心依赖
requirements_gui.txt         - GUI完整依赖（包含所有依赖）

主要依赖包：
- PyQt6>=6.4.0              (GUI界面)
- aiohttp>=3.8.0            (异步网络请求)
- beautifulsoup4>=4.11.0    (HTML解析)
- pandas>=1.5.0             (数据处理)
- python-docx>=0.8.11       (Word文档生成)
- lxml>=4.9.0               (XML解析)
- openpyxl>=3.0.0           (Excel处理)

    自动生成目录（运行时创建）

========================================
    安装步骤（3步完成）
========================================

第1步：安装 Python 3.8+
- 访问 https://www.python.org/downloads/
- 下载并安装 Python 3.8 或更高版本
- ⚠️ 重要：安装时勾选 "Add Python to PATH"

第2步：运行安装脚本
- 双击 "一键安装.bat"
- 等待安装完成（5-10分钟）
- 脚本会自动安装所有依赖并启动程序

第3步：开始使用
- 程序启动后点击 "开始爬取"
- 等待爬取完成
- 查看生成的报告和下载的文档

========================================
    日常使用
========================================

启动程序：
- 双击 "启动程序.bat"

添加网址：
- 在控制台输入网址和名称，点击"添加"
- 或在"网址管理"标签页批量导入CSV

开始爬取：
- 点击"开始爬取"按钮
- 查看实时进度

查看结果：
- Word报告：output/word_reports/
- 下载文档：output/documents/
- HTML报告：output/reports/
