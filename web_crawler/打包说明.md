## 📁 打包步骤

### 步骤1：清理临时文件

```bash
# 删除所有缓存和临时文件
cd web_crawler

# 删除 Python 缓存
rmdir /s /q __pycache__
rmdir /s /q core\__pycache__
rmdir /s /q config\__pycache__
rmdir /s /q storage\__pycache__

# 删除日志文件（可选）
del /q logs\*.log

# 删除旧的测试数据（可选）
rmdir /s /q output
```

### 步骤2：准备安装包

创建压缩包：

```bash
# 使用 7-Zip 创建压缩包
7z a -tzip 比赛通知爬虫_v2.0.zip web_crawler\

# 或使用 Windows 自带压缩
# 右键点击 web_crawler 文件夹 → "发送到" → "压缩(zipped)文件夹"
```

### 步骤3：测试安装包

1. **在新环境测试**
   - 复制到干净的 Windows 虚拟机
   - 或复制到另一台电脑

2. **测试安装流程**
   - 双击 `一键安装.bat`
   - 验证是否自动安装依赖
   - 验证是否自动启动程序

3. **测试基本功能**
   - 添加测试网址
   - 开始爬取
   - 查看输出结果

## 🚀 分发方式

### 方式1：压缩包分发（推荐）

**适用场景**：个人用户、小团队

**步骤**：
1. 按上述步骤打包
2. 通过邮件/网盘/微信发送
3. 接收方解压后双击 `一键安装.bat`

**优点**：
- 简单直接
- 无需网络（依赖已下载）
- 绿色免安装

### 方式2：Git 仓库分发

**适用场景**：开发团队、需要频繁更新

**步骤**：
1. 创建 Git 仓库
2. 上传代码
3. 用户克隆仓库运行安装脚本

**命令**：
```bash
git clone https://github.com/yourname/web_crawler.git
cd web_crawler
一键安装.bat
```

### 方式3：打包成 EXE（高级）

**适用场景**：不需要 Python 环境的用户

**步骤**：
1. 使用 PyInstaller 打包
2. 生成独立的 exe 文件

**命令**：
```bash
pip install pyinstaller
pyinstaller --onefile --windowed app_gui.py
```

**注意**：
- 文件体积会很大（>100MB）
- 某些功能可能受限
- 杀毒软件可能误报
