# 比赛通知爬虫桌面版 - 完整使用指南

## 功能概览

桌面版集成了所有功能，提供可视化操作界面：

✅ **控制台** - 一键开始/停止爬取，实时查看日志  
✅ **网址管理** - 添加/删除/导入/导出监控网址  
✅ **定时任务** - 设置自动爬取周期（每天/每周/每小时）  
✅ **结果查看** - 快速打开HTML报告和PDF文件夹  
✅ **设置** - 自定义输出目录、超时时间等参数  

---

## 安装方式

### 方式1：直接运行Python（推荐开发者）

```bash
# 1. 安装依赖
pip install -r requirements_gui.txt

# 2. 启动程序
python 启动爬虫.py
# 或
python app_gui.py
```

### 方式2：打包成EXE（推荐普通用户）

```bash
# 1. 安装打包依赖
pip install pyinstaller

# 2. 执行打包脚本
python build_exe.py --portable

# 3. 查看输出
dist/比赛通知爬虫.exe           # 独立可执行文件
dist/比赛通知爬虫_便携版/       # 便携版文件夹
```

## 使用说明

### 首次使用

1. 双击 `比赛通知爬虫.exe` 启动程序
2. 在"网址管理"标签页添加要监控的网址
3. 在"控制台"点击"开始爬取"
4. 等待爬取完成
5. 在"结果查看"标签页查看报告和PDF

## 打包部署

### 打包成单个EXE文件

```bash
python build_exe.py
```

输出：`dist/比赛通知爬虫.exe`

### 打包成便携版（推荐）

```bash
python build_exe.py --portable
```

输出：`dist/比赛通知爬虫_便携版/`

便携版包含：
```
比赛通知爬虫_便携版/
├── 比赛通知爬虫.exe      # 主程序
├── 启动程序.bat          # 启动脚本
├── 使用说明.txt          # 说明文档
├── urls.csv             # 网址列表
└── [空文件夹]
    ├── pdfs/            # PDF输出目录
    ├── reports/         # 报告输出目录
    └── data/            # 数据文件目录
```

## 常见问题

### Q1: 启动时提示缺少PyQt6
**解决**: 运行 `pip install PyQt6` 或在启动时选择自动安装

### Q2: 打包后无法运行
**解决**: 确保使用 `--add-data` 包含了所有必要的数据文件

### Q3: 定时任务不生效
**解决**: 程序需要保持运行状态，最小化到系统托盘即可

### Q4: PDF下载失败
**解决**: 启用"动态渲染"选项，或手动下载后放到pdfs目录
