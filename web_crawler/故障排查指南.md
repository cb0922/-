# 爬虫故障排查指南

## 问题描述
"Crawl failed with code 1" - 126/129个网址失败

---

## 可能原因及解决方案

### 原因1：某些特定URL导致崩溃
**症状：** 前几批成功，到某个特定URL时崩溃

**解决方案：**

1. 使用分批爬取：
```bash
python crawl_batch.py --batch-size 10
```

2. 如果某一批失败，记录批次号，从下一批继续：
```bash
# 假设第5批失败，从第6批开始
python crawl_batch.py --batch-size 10 --start-batch 6
```

---

### 原因2：内存不足（129个网址太多）
**症状：** 爬到一半程序无响应或崩溃

**解决方案：**

1. 使用更小的批次：
```bash
python crawl_batch.py --batch-size 5
```

2. 或者逐条爬取：
```bash
python crawl_batch.py --batch-size 1
```

---

### 原因3：某个网址返回异常内容
**症状：** 错误信息中有特定URL，在该URL处崩溃

**解决方案：**

1. 先清理失败记录：
```bash
del urls_fail_records.json
```

2. 修改 `enhanced_crawler_v3.py`，在 `crawl_single` 方法中添加更多异常处理（已更新）

3. 检查问题URL：
```bash
python test_first_url.py
```

---

### 原因4：编码问题
**症状：** 控制台显示乱码，或处理某些网页时崩溃

**解决方案：**

1. 设置UTF-8编码：
```cmd
chcp 65001
set PYTHONIOENCODING=utf-8
python app_gui.py
```

2. 或者使用命令行模式：
```bash
python enhanced_crawler_v3.py --urls urls.csv --output ./output 2>&1 | tee crawl.log
```

---

### 原因5：依赖库版本不兼容
**症状：** ImportError 或 AttributeError

**解决方案：**

1. 重新安装依赖：
```bash
pip install --upgrade -r requirements_gui.txt
```

2. 检查关键依赖版本：
```bash
python -c "import aiohttp; print(aiohttp.__version__)"
python -c "import PyQt6; print(PyQt6.__version__)"
```

## 推荐解决方案（按优先级）

### 方案A：使用分批爬取（推荐）

```bash
# 1. 先清理失败记录
del urls_fail_records.json

# 2. 使用分批爬取，每批10个网址
python crawl_batch.py --batch-size 10

# 3. 如果中断，从断点继续
python crawl_batch.py --batch-size 10 --start-batch N
```

**优点：**
- 避免内存问题
- 某批失败不影响其他批次
- 自动保存中间结果
- 可以从断点继续

### 方案B：使用命令行模式调试

```bash
# 设置环境
chcp 65001
set PYTHONIOENCODING=utf-8

# 运行爬虫并记录日志
python enhanced_crawler_v3.py --urls urls.csv --output ./output 2>&1 | tee crawl.log

# 查看最后50行日志了解错误
tail -n 50 crawl.log
```

### 方案C：禁用自动删除，逐个排查

```bash
# 禁用自动删除，查看哪些URL真正失败
python enhanced_crawler_v3.py --urls urls.csv --no-auto-remove --max-retries 1
```

## 应急方案

如果以上方法都不行，使用手动分批：

### 方法1：分割URL文件

```bash
# Windows PowerShell
# 将129个网址分成13个文件，每份10个

$lines = Get-Content urls.csv
$header = $lines[0]
$count = 0
$batch = 1

for ($i = 1; $i -lt $lines.Count; $i += 10) {
    $end = [Math]::Min($i + 9, $lines.Count - 1)
    $content = @($header) + $lines[$i..$end]
    $content | Out-File -FilePath "urls_batch_$batch.csv" -Encoding utf8
    Write-Host "Created urls_batch_$batch.csv"
    $batch++
}
```

然后逐个运行：
```bash
python enhanced_crawler_v3.py --urls urls_batch_1.csv --output ./output1
python enhanced_crawler_v3.py --urls urls_batch_2.csv --output ./output2
# ...
```

### 方法2：使用精简模式

```bash
# 只生成报告，不下载文档（减少资源占用）
python enhanced_crawler_v3.py --urls urls.csv --mode html
```
