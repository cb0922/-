## 使用示例

### 命令行使用

```bash
# 平衡模式（默认）
python enhanced_crawler_v2.py --urls urls.csv --mode all

# 隐形模式（应对严格反爬）
python enhanced_crawler_v2.py --urls urls.csv --mode all --stealth

# 自定义并发数
python enhanced_crawler_v2.py --urls urls.csv --mode all --concurrency 2

# 完整配置
python enhanced_crawler_v2.py --urls urls.csv --mode all --stealth --concurrency 2 --timeout 45
```

### GUI使用

1. 打开程序
2. 在"控制台"标签页
3. 勾选"启用隐形模式"（如果需要）
4. 调整"并发数"（1-5，数字越小越隐蔽）
5. 点击"开始爬取"

## 故障排查

### 问题1: 仍然被封禁

解决方案:
1. 启用隐形模式: `--stealth`
2. 降低并发数: `--concurrency 1`
3. 增加超时时间: `--timeout 60`
4. 使用代理IP

### 问题2: 速度太慢

解决方案:
1. 关闭隐形模式（使用平衡模式）
2. 增加并发数: `--concurrency 5`
3. 筛选响应快的网站

### 问题3: 内存占用高

解决方案:
1. 降低并发数
2. 使用流式处理（已集成）
3. 分批爬取
