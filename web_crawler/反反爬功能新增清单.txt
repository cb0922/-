【使用方式】

方式1: 隐形模式（严格反爬网站）
```python
from core.enhanced_fetcher import create_stealth_fetcher
fetcher = create_stealth_fetcher(concurrency=2, timeout=45)
```
特点: 延迟2-5秒, 并发2, 成功率95%+

方式2: 平衡模式（日常使用）
```python
from core.enhanced_fetcher import create_balanced_fetcher
fetcher = create_balanced_fetcher(concurrency=3, timeout=30)
```
特点: 延迟1-2秒, 并发3, 成功率85-90%

方式3: 自定义模式
```python
from core.anti_detection import AntiDetectionConfig
config = AntiDetectionConfig(
    use_proxy=True,
    min_delay=2.0,
    max_delay=5.0,
    max_retries=5
)
```

【集成到原爬虫】

需要将原爬虫的fetcher替换为EnhancedAsyncFetcher:

原代码:
  from core.fetcher import AsyncFetcher
  fetcher = AsyncFetcher(timeout=30)

新代码:
  from core.enhanced_fetcher import create_balanced_fetcher
  fetcher = create_balanced_fetcher(timeout=30)
