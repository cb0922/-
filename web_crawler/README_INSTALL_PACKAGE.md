# 比赛通知爬虫 v2.0 - 安装包说明

## 📦 欢迎使用比赛通知爬虫！

本安装包包含了完整的使用说明和自动化安装脚本，即使是没有 Python 基础的用户也能轻松安装使用。

---

## 🚀 快速开始（只需3步）

### 第1步：安装 Python
- 访问 https://www.python.org/downloads/
- 下载 Python 3.8 或更高版本
- **⚠️ 关键**：安装时勾选 **"Add Python to PATH"**

### 第2步：运行安装脚本
- 双击 `一键安装.bat`
- 等待 5-10 分钟自动安装依赖
- 安装完成后自动启动程序

### 第3步：开始使用
- 程序启动后点击 **"开始爬取"**
- 等待爬取完成
- 查看生成的 Word 报告和下载的文档

## 💻 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 (64位) |
| Python | 3.8 或更高版本 |
| 内存 | 建议 4GB 以上 |

## ⚡ 安装方式对比

| 方式 | 难度 | 耗时 | 适用场景 |
|------|------|------|----------|

## 🆘 常见问题速查

| 问题 | 解决方案 |
|------|----------|
| Python 未安装 | 访问 python.org 下载安装，勾选 "Add to PATH" |
| 安装速度慢 | 脚本已配置国内镜像，如仍慢请检查网络 |
| 权限不足 | 右键脚本选择"以管理员身份运行" |
| 杀毒软件拦截 | 添加到信任列表或暂时关闭 |
| 程序闪退 | 查看【首先阅读】安装与使用指南.md 的故障排除章节 |
