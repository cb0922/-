# 比赛通知爬虫 v2.0.1 - 路径问题紧急修复

## ⚠️ 重要说明

此版本修复了**跨设备路径问题**，解决了在其他设备上运行时GUI界面报错的问题。

### 问题现象
- 本地运行正常，发送到其他设备后报错
- 提示 "URL file not found: urls.csv"
- 程序无法找到数据文件

### 修复方案
- 启动脚本自动切换工作目录
- 所有文件操作使用绝对路径
- 不再依赖工作目录

## 💡 技术说明

### 为什么会出现路径问题？

**本地开发环境：**
```
工作目录: C:\Users\PC\kim\web_crawler\
程序目录: C:\Users\PC\kim\web_crawler\
结果: ✅ 正常
```

**用户双击启动环境（修复前）：**
```
工作目录: C:\Users\User\Desktop\
程序目录: C:\Program Files\爬虫\
结果: ❌ 找不到 urls.csv
```

**用户双击启动环境（修复后）：**
```
工作目录: C:\Program Files\爬虫\  (自动切换)
程序目录: C:\Program Files\爬虫\
结果: ✅ 正常
```

### 修复的核心代码

**批处理脚本：**
```batch
cd /d "%~dp0"
```

**Python程序：**
```python
self.app_dir = os.path.dirname(os.path.abspath(__file__))
urls_file = os.path.join(self.app_dir, "urls.csv")
```
