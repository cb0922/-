### ✅ 1. 多格式文档支持

**支持格式（12类）:**

| 类型 | 扩展名 |

| PDF | .pdf |

**功能:**
- 从HTML中提取所有支持的文档链接
- 自动识别文档类型并分类
- 智能判断文档是否与比赛相关
- 流式下载大文件

---

### ✅ 2. 自动删除无效网址

**机制:**

```
网址访问失败
    ↓
失败次数+1 (记录到 urls_fail_records.json)
    ↓
达到3次?
    ├─ 是 → 自动从 urls.csv 删除
    └─ 否 → 下次继续尝试
```

**配置:**
- 默认: 3次失败后删除
- 可调: `--max-retries N`
- 禁用: `--no-auto-remove`

**文件:** `enhanced_crawler_v3.py`

## 使用方式

### GUI界面
```bash
python app_gui.py
```
1. 切换到"设置"标签页配置代理（可选）
2. 选择爬取模式（all/doc/html）
3. 点击"开始爬取"

### 命令行
```bash
# 基础使用
python enhanced_crawler_v3.py --urls urls.csv

# 使用代理
python enhanced_crawler_v3.py --urls urls.csv --proxy

# 调整失败次数阈值
python enhanced_crawler_v3.py --urls urls.csv --max-retries 5

# 禁用自动删除
python enhanced_crawler_v3.py --urls urls.csv --no-auto-remove
```
