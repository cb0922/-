# 代理IP配置说明

## 什么是代理IP

代理IP是指通过一个中间服务器（代理服务器）来访问目标网站。使用代理可以：

1. **突破访问限制** - 绕过网站的IP封锁
2. **隐藏真实IP** - 避免被网站识别和封禁
3. **提高成功率** - 当一个IP被限制时自动切换到其他IP
4. **分布式爬取** - 多个代理同时进行，提高效率

## 配置文件格式

### JSON格式（推荐）

创建 `proxies.json` 文件：

```json
[
  {
    "host": "192.168.1.1",
    "port": 8080,
    "protocol": "http",
    "username": null,
    "password": null,
    "name": "代理1",
    "is_working": true
  },
  {
    "host": "proxy.example.com",
    "port": 1080,
    "protocol": "socks5",
    "username": "user",
    "password": "pass",
    "name": "付费代理",
    "is_working": true
  }
]
```

### TXT格式（每行一个）

创建 `proxies.txt` 文件：

```
# HTTP代理
http://192.168.1.1:8080
http://user:pass@proxy.example.com:8080

# HTTPS代理
https://proxy.secure.com:8443

# SOCKS5代理
socks5://127.0.0.1:1080
socks5://user:pass@socks.proxy.com:1080
```

## 支持的代理协议

| 协议 | 说明 | 示例 |
|------|------|------|
| HTTP | 普通HTTP代理 | `http://host:port` |
| HTTPS | 加密HTTP代理 | `https://host:port` |
| SOCKS5 | SOCKS5代理 | `socks5://host:port` |

## GUI界面配置

1. 打开程序，切换到"设置"标签页
2. 勾选"使用代理IP"
3. 选择代理类型
4. 指定代理文件路径（默认 `proxies.json`）
5. 点击"添加代理"手动添加代理
6. 点击"测试代理"验证代理可用性
7. 点击"代理状态"查看统计信息

## 命令行使用

```bash
# 使用默认代理文件 (proxies.json)
python enhanced_crawler_v2.py --urls urls.csv --proxy

# 指定代理文件
python enhanced_crawler_v2.py --urls urls.csv --proxy --proxy-file my_proxies.json
```

## 代理自动管理

爬虫会自动：

1. **轮换代理** - 每次请求使用不同代理
2. **失败标记** - 连续失败3次的代理会被标记为不可用
3. **自动重试** - 当代理失败时自动切换到其他代理
4. **延迟测试** - 可手动测试代理延迟

## 故障排除

### 代理测试失败

1. 检查代理格式是否正确
2. 验证代理是否需要用户名密码
3. 确认代理服务器是否在线
4. 检查本地防火墙设置

### 所有代理都失败

1. 检查网络连接
2. 验证代理文件格式
3. 尝试不使用代理直接访问
4. 联系代理提供商

### 爬虫速度变慢

1. 代理本身速度较慢
2. 每个请求增加了代理转发时间
3. 建议：使用响应速度快的付费代理

## 安全提示

1. **不要在代码中硬编码代理密码**
2. **定期更换代理密码**
3. **使用HTTPS代理保护数据传输**
4. **选择信誉良好的代理提供商**

## 示例：批量添加代理

```python
from core.proxy_manager import ProxyManager

manager = ProxyManager("proxies.json")

# 批量添加
proxies = [
    "http://192.168.1.1:8080",
    "http://user:pass@192.168.1.2:8080",
    "socks5://127.0.0.1:1080",
]

for p in proxies:
    manager.add_proxy_from_string(p)

# 测试所有代理
import asyncio
asyncio.run(manager.test_all_proxies())
```
